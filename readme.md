## Next.js Cookie Auth

Experimenting with cookies in Next.js

Following this [udemy tutorial](https://www.udemy.com/universal-react-with-nextjs-the-ultimate-guide).

## Install & Run

1. `git clone https://github.com/JollyGrin/next-cookie-auth.git`
2. `npm i`
3. `npm run dev`
